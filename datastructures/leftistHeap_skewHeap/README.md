# Leftist Heap and Skew Heap
Goal: Implement a leftist heap and skew heap data structure to show benefits
of insert and remove time complexity <br />

## How to Run
Requires: make <br />

### Assemble
make LeftistSkew

### Run Leftist Heap
./LeftistSkew leftist

### Run Skew heap
./LeftistSkew skew

### Remove .o and executables
make clean
